# android_device_honor_msm8937_64
### Recovery Tree.

OrangeFox device tree for Honor 7A (AUM-L29)
- To build OrangeFox, use branch Android 11
- Clone it to device/honor/msm8937_64
- Build with lunch `twrp_msm8937_64-eng`.
